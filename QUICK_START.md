# Quick Start Guide

## Installation (3 Simple Steps)

1. **Open Chrome Extensions**
   - Go to: `chrome://extensions/`
   - Turn ON "Developer mode" (top-right toggle)

2. **Load Extension**
   - Click "Load unpacked"
   - Select this folder: `chrome ext`

3. **Use It!**
   - Click the extension icon in Chrome toolbar
   - (Optional) Pin it for quick access

## Usage

### Timer
1. Enter hours, minutes, seconds
2. Click Start
3. Get notified when done!

### Stopwatch
1. Click Start to begin counting
2. Pause/Resume anytime
3. Reset to clear

### Pomodoro
1. Click ⚙️ Settings to customize durations
2. Click Start to begin work session
3. Auto-transitions between work and breaks
4. Completes 4 cycles, then long break

## Tips

- Timers continue even if you close the popup
- Works offline (no internet needed)
- Notifications show when sessions complete
- All settings are saved automatically

## Troubleshooting

**Extension won't load?**
- Make sure all files are in the folder
- Check for errors at `chrome://extensions/`

**Timer doesn't persist?**
- Verify "storage" permission in manifest.json
- Check service worker is running

**No notifications?**
- Enable Chrome notifications in system settings
- Check browser notification permissions
